# Plane — Product Context

Plane is an open-source project management tool for software teams (https://github.com/makeplane/plane).

## Core concepts

- **Workspaces** — top-level org container
- **Projects** — group issues within a workspace
- **Issues** — tasks with states (Backlog, Todo, In Progress, Done, Cancelled)
- **Cycles** — time-boxed sprints
- **Modules** — feature groupings across issues
- **Views** — saved filters and layouts (List, Kanban, Calendar)

## UX conventions

- Left sidebar navigation: workspace → projects → issues
- Issue detail opens in a side panel or full page
- State changes via dropdown or keyboard shortcuts
- Assignees, labels, priorities on issues
- Comments and activity feed on each issue

## Target users

- Engineering teams (5–50 people)
- Product managers tracking delivery
- Team leads managing cycles and capacity
