# UX Research Skill

## Quick Start
1. Read `philosophy-principles.md`
2. Read `research-classification.md`
3. Use `planning/` to frame work
4. Use `methods/` to run studies
5. Use `analysis-synthesis/` and `impact-delivery/` to drive decisions

## Bundle Map
- Core: philosophy + classification
- Planning: templates + stakeholder alignment
- Methods: interviewing, evaluative, generative, quant, advanced
- Synthesis: tagging, clustering, frameworks, topline
- Delivery: tracker, stakeholder engagement, ops, maturity
- Assets: templates, checklists, prompts, references
