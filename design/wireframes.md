# Wireframes for Healthcare Portal

## Overview of Components
- **Homepage**: A clean layout with large buttons for Sign In and Learn More. Includes alt text for images and keyboard navigation features.
- **Sign In Page**: Fields for username and password, with labels. Clear instructions on keyboard navigation.
- **Appointment Booking Page**: Simple calendar view for booking, clearly labeled options, with confirmation buttons that are keyboard accessible.
- **Results List Page**: Accessible table showcasing results with filters and sort options that comply with WCAG rules.

## Accessibility Features
- Use of ARIA roles for buttons, alerts, and navigation menus.
- High-contrast colors and legible fonts for readability.
- Captioning for audiovisual content.

## Notes
- Incorporate user feedback into the design.
- Regularly check for compliance with WCAG 2.2 AA standards.
