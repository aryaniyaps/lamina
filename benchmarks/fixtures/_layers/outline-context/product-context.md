# Outline — Product Context

Outline is an open-source team knowledge base (https://github.com/outline/outline).

## Core concepts

- **Collections** — folders organizing documents
- **Documents** — rich-text collaborative pages with real-time editing
- **Templates** — reusable document structures
- **Search** — full-text across all accessible docs
- **Permissions** — collection-level and document-level access control

## UX conventions

- Sidebar: collections tree + recent/starred docs
- Editor: Notion-like blocks, slash commands, @mentions
- Document header: title, share button, presence avatars
- Comments inline on text selections
- Dark mode support

## Target users

- Internal documentation authors
- Team leads organizing knowledge
- New hires onboarding via docs
