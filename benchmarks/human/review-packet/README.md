# Blind human evaluation packet

Generated by `npm run bench:human-packet`.

## Instructions for raters

1. You will review pairs of **product design artifacts** (labeled **A** and **B**) for each task
2. You do **not** know which arm (with/without Lamina) produced which artifact
3. Score each artifact independently using [`../../judges/rubric.md`](../../judges/rubric.md) — product-behavior criteria, not UX research docs
4. Submit scores via the CSV template in this directory

## Rubric criteria (1–5 each)

1. Domain / system structure
2. Invariants and product rules
3. Actors and permissions
4. Workflow quality
5. Scenario / edge coverage
6. Systems judgment
7. UX expression under rules
8. Brownfield fit
9. Implementation readiness
10. Overall product-behavior quality

## Files

- `packet.html` — Browser-friendly review UI
- `scores-template.csv` — Blank scoring sheet
- `answer-key.json` — Arm mapping (do not open until scoring is complete)
