## Instructions for raters

Methodology: **[Design A — ecological adoption](../../METHODOLOGY.md)**.

1. Review pairs of **implementations** (labeled **A** and **B**) — bundled application source
2. You do **not** know which arm produced which implementation
3. Score using [`../../judges/rubric.md`](../../judges/rubric.md) if providing feedback
4. Submit via CSV (`npm run bench:import-human`) — maintainers may attach as an appendix in `report.md`
