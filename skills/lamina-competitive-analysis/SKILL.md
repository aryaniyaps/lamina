# Reference Patterns (agent-native)

Use **user-provided references** (URLs, screenshots, named products) to inform patterns — never autonomous market research decks.

## Allowed

- "User cited Notion sidebar — adopt similar IA for settings"
- Walkthrough of public demo if user points to it
- Explicit non-goals ("we are not copying their paywall")

## Forbidden

- Autonomous competitive landscape reports
- Parity featuritis without invariant justification

## Output

Brief `decisions.md` entry: reference, pattern borrowed, deliberate differences.
