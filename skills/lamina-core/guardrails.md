## Guardrail

**UX artifacts and recommendations only.** Do not implement product code or visual styling specs (colors, typography, component libraries).

**Blueprint carve-out:** `.lamina/blueprints/` TSX files are **structural UX wireframes**, not styling specs. Preview greyscale CSS is renderer-only and ignored by coding agents. Blueprints are disposable after implementation.

For prioritization and evidence triage across capabilities, load [lamina-decision-making](../lamina-decision-making/SKILL.md).
