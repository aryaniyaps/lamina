# Design how it works.

Lamina is a headless product-design skill for AI coding agents. It specs how your app **works** — states, roles, flows, edge cases, permission matrices — into a contract your agent implements, then verifies your live build with parallel persona walks.

UI skills dress the app. Lamina designs how it behaves.

## Who it's for

Lamina is for developers who build with AI coding agents (Cursor, Claude Code, Codex, Gemini, Pi) and want product behavior designed before app code ships. You keep your stack, UI library, and agent — Lamina adds a design → implement → verify loop that catches impossible states, skipped dependencies, and dead-end flows that UI-only skills miss.

**Outcomes you should expect:**

- A machine-readable contract (`run.yaml`) your agent implements from `implement.md`
- Parallel persona walks that surface product gaps before users do
- A fix loop (`fix.md` → agent fixes → re-verify) until the contract holds

See the [HavenStay demo](/guides/demo-havenstay) for a side-by-side build with and without Lamina.

## Quick path

```bash
npx skills install aryaniyaps/lamina
```

```
/lamina-init <your domain>
/lamina-design <feature or flow>

# your agent implements from implement.md
/lamina-verify http://localhost:3000
```

## The loop

```text
/lamina-init     →  business-context.md + personas.yaml
/lamina-design   →  run.yaml + implement.md (ready_to_build)
       ↓
  Your agent implements (any stack)
       ↓
/lamina-verify   →  persona walks → fix.md when needed
       ↓
  Your agent fixes → /lamina-verify again until contract holds
```

## What Lamina is not

- **Not a UI library** — Lamina never writes app source or picks your component library
- **Not an app builder** — it does not generate stack-locked apps like v0 or Lovable
- **Not a pixel skill** — Impeccable and frontend-design polish how it looks; Lamina designs how it works

## Fits your stack

Lamina slots into whatever you already use — any AI coding tool, framework, database, language, and UI library. See [Any stack](/concepts/what-is-lamina#any-stack).
