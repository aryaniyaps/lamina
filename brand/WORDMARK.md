# Lamina Wordmark Spec v3

## Lockup

```
[layer mark]  lamina
```

Mascot is separate — never in the logo lockup. Domain `lamina.dev` is for URLs only, not the wordmark.

## Typography

| Element | Font | Weight |
|---------|------|--------|
| `lamina` | Geist Sans | 600 |

- Letter-spacing on `lamina`: `-0.04em`
- Always lowercase — never `Lamina` or `LAMINA` in product lockup
- No `.dev` suffix in the wordmark — cleaner lockup; use full domain in URLs and footers only

## Rules

- Flat type only — no bevel, gradient, or 3D on wordmark
- No custom dot on `i` — standard typography
- Mark = structural (leaf / L / wireframe cards); personality lives in 3D meerkat elsewhere
